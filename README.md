# LinkedIn Profile Setup

Turn any AI chat into a LinkedIn profile interviewer for students. Paste one file in, answer some questions, get a complete, click-by-click guide to your own profile back.

## What this is

Most LinkedIn advice online is generic and written for people already deep into a career. This is different: a single, portable `.md` file that works as a system prompt for any AI chat assistant (Claude, ChatGPT, Gemini, or anything else you can paste text into). It interviews a student about who they are, what they've built, and what they've achieved, then hands back a section-by-section guide to their actual profile, not a template, not filler advice, an ordered checklist of exactly what to click and what to type.

It's built from a real, hands-on profile edit, refined section by section until each part held up to real scrutiny: a mistagged skill caught and fixed, an About section that read as obviously AI-written and got rewritten until it didn't, a skill wrongly credited to a school that never taught it, a headline stripped of a namedrop that wasn't actually a differentiator. The rules in this file exist because a real mistake surfaced each one, not because they sounded reasonable in the abstract.

## How to use it

1. Open `SKILL.md`.
2. Copy the entire file.
3. Paste it as your first message into any AI chat.
4. Answer the questions it asks you honestly and specifically, the more detail you give, the better the output.
5. Get back an ordered Markdown checklist covering your whole profile, ready to work through top to bottom (manual)

No LinkedIn or career-coaching knowledge required going in. That's the point.

## What it covers

Headline, About, Education, Skills, Honors & Awards, Experience/Position (including how to frame independent or self-taught work honestly), Projects, Contact Info, and Featured, in the order they actually appear on a profile.

## Why it's built this way

- **Interview first, write second.** The AI never guesses at your background, it asks, every time, before generating anything.
- **Handles zero achievements or twenty, equally.** Every achievement/title/project question is open-ended. Some students have one thing to list, some have many. Neither is treated as the default.
- **Honesty is a hard rule, not a style choice.** No fabricated numbers, no overclaiming unfinished work as live, no borrowing credibility from an institution that didn't actually teach a skill.
- **Confidentiality-aware.** It checks before writing about anything that might involve a client, employer, or private agreement.
- **Anti-AI-tell writing bar.** No em dashes, no corporate vocabulary, no formulaic bullet-header structure that instantly reads as machine-written.
- **Portable by design.** No dependence on any specific AI platform's tools or features. Plain text in, plain Markdown out, works everywhere.
